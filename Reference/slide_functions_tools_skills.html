MDLC Agentic AI Platform

Key Functions, Tools & Skills

The platform operates through three cooperating capability types: Skills shape reasoning. MCP Tools expose authoritative system actions. Deterministic SDKs perform all material execution.
Separation of Concerns · Governance-First · Token-Thrifty

Use MCP tools when the capability...
Changes or reads authoritative system state
Touches governance, registry, approvals, or artifacts
Requires a strict schema-validated payload
Must be auditable and deterministic
Must be shared across agents or runtimes
Involves manifest, lineage, or model product operations
Use skills when the capability...
Is primarily a reasoning or review pattern
Is a drafting, checklist, or interpretation guide
Does not own or mutate authoritative state
Shapes how the agent thinks, not what it records
Can be reused across agents and model families
Benefits from compact, token-thrifty instruction
